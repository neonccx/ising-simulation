Animation: 20×20 lattice evolving as temperature increases from T = 1.0 to T = 10.0 (β from 1.0 → 0.1). Left: spin configuration (red = +1, blue = −1). Center: order parameter ⟨|M|⟩/N². Right: mean energy ⟨E⟩/N². The phase transition near Tc ≈ 2.269 is clearly visible.

System

Lattice: 20 × 20
Spins: si = ±1
Boundary: open

Parameters

Coupling: J = 1
Boltzmann: kB = 1
β range: 0.1 → 1.0

Algorithm

Metropolis MC
Checkerboard update
400 equilibration sweeps

Critical Point

Onsager exact:
Tc = 2/ln(1+√2)
≈ 2.2692

Hamiltonian: E = −J Σ si sj (sum over nearest-neighbor pairs)
Metropolis acceptance: accept flip if ΔE ≤ 0; otherwise accept with probability exp(−β ΔE)
Phase transition: below Tc the system spontaneously breaks Z₂ symmetry → ferromagnetic order. Above Tc thermal fluctuations destroy long-range order → paramagnetic phase.
